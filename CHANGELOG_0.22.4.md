# 版本 0.22.4 更新日志

## 🐛 Bug 修复

### 修复 Workspace 切换时侧边栏空白问题

**问题**：当用户在 Obsidian 中切换 workspace 时，Vertical Tabs 侧边栏显示为空白区域，无法显示标签页内容。

**修复内容**：

1. **增强视图渲染机制**
   - 添加渲染状态追踪，确保组件正确初始化
   - 实现智能重新渲染，当检测到视图未正确显示时自动恢复
   - 优化 React 组件的挂载和卸载流程

2. **添加 Workspace 变化监听**
   - 监听 layout-change 事件，及时响应 workspace 切换
   - 在检测到视图异常时触发强制刷新

3. **增强可见性检查**
   - 在组件挂载时检查视图是否真正可见
   - 延迟刷新机制确保 DOM 完全加载后再渲染

**影响范围**：
- 修复了 workspace 切换功能
- 不影响现有功能
- 提升了视图稳定性

**测试方法**：
1. 创建多个 workspace
2. 在不同 workspace 之间切换
3. 验证侧边栏是否正常显示标签页
