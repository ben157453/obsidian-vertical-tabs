# Changelog v1.1.7

## 新增功能 (New Features)

### 跨组投掷功能 (Cross-Group Link Throwing)
- 🎯 **核心功能**：新增链接跨组投掷功能，允许用户通过修饰键将链接直接投掷到 FGroup 内的下一个子组
- ⌨️ **操作方式**：
  - **Ctrl + 点击**：链接在当前 FGroup 的下一个子组打开，视图焦点跟随到新标签页
  - **Shift + 点击**：链接在当前 FGroup 的下一个子组打开，但视图焦点留在原地
- ⚙️ **灵活配置**：
  - 可在设置中启用/禁用跨组投掷功能
  - 可选择是否需要修饰键触发（支持无修饰键全自动投掷模式）
- 🚀 **性能优化**：
  - 采用三层寻址策略提升稳定性
  - 优先使用 tabCacheStore 缓存提高查找效率
  - 回退机制确保在各种情况下都能准确定位目标子组

## 技术改进 (Technical Improvements)

### 寻址算法优化
- 实现健壮的三层寻址机制：
  1. **缓存直连**：优先从 tabCacheStore 获取分组引用
  2. **反向追溯**：遍历所有叶子节点寻找目标分组
  3. **全局搜索**：递归搜索整个工作区树作为备选方案

### 错误处理增强
- 添加详细的错误提示和日志记录
- 边界情况检查和优雅降级处理
- 用户友好的通知消息

## 设置选项 (Settings Options)

在 Miscellaneous 设置区域新增：
- **Enable link throwing**：启用跨组投掷功能
- **Require modifier key**：要求使用修饰键触发（当功能启用时可见）
